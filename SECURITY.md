# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

## Reporting a vulnerability

Email s@scrrlt.dev with a description of the issue, reproduction steps, and any
available logs. You will receive an acknowledgement within two business days.

## Disclosure

Security fixes are published as patch releases. Advisories are posted through
GitHub’s security channels when applicable.
